# 🛒 Shopping Cart

A simple, interactive shopping cart web application built with vanilla HTML, CSS, and JavaScript.

## 🌐 Live Demo

**[View the live app here →](https://garychamp.github.io/shopping-cart/)**

---

## 📋 Features

- **View cart items** – Displays a list of items with their name, price, and quantity
- **Add items** – Add new items to the cart via an input form (name, price, quantity)
- **Increase quantity** – Increment the quantity of any item with the `+` button
- **Remove items** – Delete items from the cart with the `Remove` button
- **Live total** – The cart total updates automatically whenever items are added, removed, or changed
- **Pre-loaded items** – Cart comes pre-populated with sample grocery items (Milk, Bread, Eggs)

## 🚀 Getting Started

No build tools or dependencies required. Just open the project in your browser.

### Run locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Garychamp/shopping-cart.git
   ```
2. Navigate into the project directory:
   ```bash
   cd shopping-cart
   ```
3. Open `index.html` in your browser.

## 💡 How to Use

1. The cart loads with a few pre-filled items.
2. To **add a new item**, enter the item name, price, and quantity in the form fields and click **Add Item**.
3. To **increase the quantity** of an item, click the **`+`** button next to it.
4. To **remove an item**, click the **`Remove`** button next to it.
5. The **total** at the bottom updates automatically in real time.
